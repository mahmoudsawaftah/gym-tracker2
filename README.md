# Iron Log — برنامجي التدريبي

تطبيق ويب من ملف واحد (بدون أي build step) لمتابعة برنامجك الأسبوعي: 6 أيام تدريب، فيديو شرح لكل حركة، تسجيل الوزن والعدات، مؤقت راحة، وحلقة تقدم أسبوعية.

## رفعه على GitHub Pages (5 دقائق)

1. اعمل ريبو جديد على GitHub، مثلاً `gym-tracker`.
2. ارفع ملف `index.html` هذا لجذر الريبو.
3. من إعدادات الريبو → Pages → اختار Branch: `main` / Folder: `/root` → Save.
4. بعد دقيقة، الموقع رح يشتغل على:`https://<username>.github.io/gym-tracker/`

## ملاحظات مهمة

* **التخزين محلي (localStorage)**: كل بيانات التسجيل (الوزن، العدات، الحركات المنجزة) بتنحفظ بمتصفحك بس. إذا بدك مزامنة بين أجهزة، هاي أول خطوة لازم نضيفها لاحقًا (Supabase).
* **علامة "🔍 بديل موثوق"**: يعني الفيديو مو من قناة DeltaBolic مباشرة (لأنه ما إله فيديو مخصص لهالحركة)، والرابط بيوديك لبحث داخل قناة موثوقة (Jeff Nippard). افتحه واختار أفضل نتيجة، وبعدين بلّشني احدّث الرابط المباشر.
* **علامة "✓ مصدر مؤكد"**: رابط فيديو محدد وتم التحقق إنه من قناة DeltaBolic (يوتيوب أو تيك توك).
* الحركات المعلّم عليها ⚠️ بالملاحظات مرتبطة بمنطقة الكوع — راقب حسك فيها بسبب انزعاجك بالكوع اليمين.

## الخطوة الجاية (لو بدك التوسع)

* ربط Supabase لحفظ البيانات بالسحابة + حساب مستخدم.
* صفحة "التقدم" برسوم بيانية (حجم تدريب أسبوعي، PR لكل حركة).
* نظام Progressive Overload تلقائي يقترح زيادة الوزن حسب آخر جلسة.
* تحويله لـ PWA (يشتغل أوفلاين + يتثبت على الشاشة الرئيسية).

هاي كلها ما زادتها بهالنسخة عشان توصلك نسخة شغالة وقابلة للرفع فورًا، بدل ما تضل منتظر "المشروع الكامل" أسابيع.
